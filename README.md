# AI Code Review System
This project is an AI-based code review system that analyzes source code and gives suggestions to improve code quality.

The system uses Ollama with the Qwen model to review code locally without using any external API key.
## Features
- Automatic code review
- Detects bugs and issues
- Suggests improvements
- Works locally using Ollama
- No API key required
- ## Technologies Used
- Python
- Ollama
- Qwen LLM
- HTML
- CSS
- JavaScript
