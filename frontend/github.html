📋 Setup Instructions

1

Open GitHub Repository Settings

Navigate to your repository on GitHub and click Settings > Webhooks

2

Add Webhook

Click "Add webhook" button

3

Configure Webhook

application/json

Set environment variable GITHUB_WEBHOOK_SECRET on your server

4

Select Events

Choose "Just the push event"

5

Activate Webhook

Make sure "Active" is checked and click "Add webhook"
